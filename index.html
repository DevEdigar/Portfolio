Bootstrap

Bootstrap é um framework front-end que simplifica a criação de sites responsivos e estilizados. Oferece um sistema de grid que se adapta a diferentes tamanhos de tela e uma ampla gama de componentes prontos, como botões e formulários. Com um design moderno e personalizável, Bootstrap ajuda a acelerar o desenvolvimento de páginas web, garantindo uma aparência consistente em diversos navegadores e dispositivos.

HTML e CSS

HTML (HyperText Markup Language) é a linguagem padrão para criar e estruturar páginas da web. Ele serve como o "esqueleto" de um site, permitindo que desenvolvedores organizem o conteúdo e definam a estrutura básica das páginas. HTML utiliza tags (ou elementos) para marcar textos, imagens, links, vídeos e outros tipos de conteúdo, determinando como eles serão exibidos no navegador.CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas da web, controlando a aparência dos elementos HTML. Ele define aspectos visuais como cores, fontes, espaçamentos e layouts. Utiliza seletores para aplicar estilos a elementos específicos e propriedades para definir regras como cor e tamanho da fonte. CSS permite criar páginas web visualmente atraentes e responsivas, separando o conteúdo (HTML) do design.

Criação do Portfólio

Este portfólio foi desenvolvido como projeto prático durante minha graduação em Análise e Desenvolvimento de Sistemas, com o objetivo de criar um espaço profissional para apresentar meus projetos e habilidades em desenvolvimento front-end. Utilizei um template Bootstrap como base, personalizando-o extensivamente com HTML5, CSS3 e JavaScript para criar uma apresentação única e profissional do meu trabalho. O design foi totalmente remodelado, com a escolha de uma paleta de cores moderna e uma tipografia legível, visando uma experiência de usuário agradável e intuitiva. A responsividade foi uma prioridade, garantindo que o portfólio se adaptasse perfeitamente a diferentes dispositivos, desde desktops até smartphones. Além das seções básicas de apresentação e projetos, adicionei funcionalidades extras, como um formulário de contato dinâmico e uma galeria de imagens interativa.
